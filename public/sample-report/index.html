Quality check videos, podcasts, and clips before you upload.

Verdict examples

  • PASS: clean upload evidence.
  • WATCH: fixable caption or layout warnings.
  • BLOCK: source/render issues before publishing.

Public report examples

PASS

Clean upload

No hard quality gates fired. The report records checked minutes, cost estimate, source metadata, and the exact checks that ran.

WATCH

Caption or layout warning

Example: caption text sits close to a Shorts UI safe area. Agent can show the flag and ask whether to move or restyle the overlay before rerunning QC.

BLOCK

Duplicate crowd characters

Example: repeated faces and robe silhouettes appear across a crowd scene. UploadCheck tells the editor to re-render with distinct character variation before upload.
