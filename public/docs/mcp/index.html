UploadCheck.app
Overview MCP API Sample report

MCP reference

Install command for public agent workspaces:

npx -y @drantoniou/uploadcheck-mcp

Set UPLOADCHECK_API_BASE_URL=https://api.uploadcheck.app and UPLOADCHECK_API_KEY=<workspace_api_key>.

qc_get_launch_status

Fetch live UploadCheck launch readiness, current blockers, and operator commands for agent planning.

Inputs

none

Outputs

product_hunt_ready, status, remaining_blockers, operator_commands

qc_get_launch_handoff

Fetch live UploadCheck launch blockers, required actions, proof commands, and no-launch rule for agent/operator handoff.

Inputs

none

Outputs

productHuntReady, remainingBlockers, requiredActions, blockerProofCommands, launchDoctorCommands, operatorCommandSequence, rule

qc_get_launch_doctor

Fetch live UploadCheck launch blocker fix plan for Product Hunt readiness.

Inputs

none

Outputs

productHuntReady, remainingBlockers, blockerFixPlan, blockerProofCommands, launchDoctorCommands, operatorCommandSequence, rule

qc_get_launch_evidence

Fetch redacted UploadCheck launch evidence derived from the live launch doctor for agent-safe Product Hunt verification.

Inputs

none

Outputs

productHuntReady, status, blockers, commandCoverage, fixPhases, redaction

qc_get_pipeline_handoff

Fetch the production pipeline handoff runbook for NTO/NPO and creator workflows.

Inputs

none

Outputs

source_artifacts, profiles, call_sequence, media_ingress, margin_rules, repair_loop_contract

qc_get_npo_pipeline_handoff

Fetch the focused NPO podcast/audio production handoff for cost preflight, sidecars, media ingress, marker CSV, and repair-loop workflows.

Inputs

none

Outputs

profile_id, cost_preflight, media_ingress, required_sidecars, mcp_sequence, repair_loop_contract

qc_get_pipeline_recipes

Fetch machine-readable UploadCheck pipeline profiles and repair-loop contracts for NTO/NPO and creator workflows.

Inputs

none

Outputs

launch_preflight, cost_preflight, profiles, repair_loop_contract, nto_replacement_qc

qc_get_cost_basis

Fetch public cost-per-minute, plan economics, and 95% gross-margin assumptions before pricing or launch decisions.

Inputs

none

Outputs

target_gross_margin_pct, cost_assumptions, plans, verdict, model_backed_checks

qc_estimate_cost

Preflight cost, margin safety, and effective/removed gates before an agent uploads or runs media.

Inputs

minutes, duration_seconds, checks, plan_id, ai_review_seconds, cost_guardrail

Outputs

margin_safe, effective_checks, removed_checks, cost_estimate, action

qc_run_video

Start a full-timeline UploadCheck run from Claude, Codex, or another agent workspace.

Inputs

youtube_url, upload_id, signed_url, video_base64, audio_base64, data_url, manifest_url, transcript_url, watchlist_url, expected_script_url, chunk_sidecars_url, callback_url

Outputs

job_id, verdict, status_url, report_url, minutes_metered, media_ingress, sidecar_ingress, cost_estimate

qc_run_local_file

Read a local media file from an agent workspace and send it through Render inline or signed upload.

Inputs

file_path, checks, manifest_path, transcript_path, watchlist_path, expected_script_path, sidecar_dir, plan_id, cost_guardrail

Outputs

job_id, verdict, status_url, report_url, minutes_metered, media_ingress, cost_estimate

qc_get_job

Inputs

job_id

job_id

Outputs

events, event_type, created_at

qc_get_marker_csv
