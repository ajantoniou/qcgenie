REST API reference

All 27 UploadCheck endpoints.

Every request authenticates with Authorization: Bearer <workspace_api_key> against https://api.uploadcheck.app. The full machine-readable spec lives at /openapi.json.

Base

  • Host: api.uploadcheck.app
  • Auth: Bearer <workspace_api_key>
  • Spec: /openapi.json
  • Manifest: /agent-manifest.json

Endpoints

GET/v1/launch-status

Fetch live Product Hunt launch go/no-go state, blockers, and operator commands.

GET/v1/launch-handoff

Fetch live Product Hunt launch blockers, required actions, proof commands, and no-launch rule.

GET/v1/launch-doctor

Fetch live Product Hunt blocker fix plan and normalized launch-doctor command coverage.

GET/v1/launch-evidence

Fetch redacted live launch evidence for agent-safe Product Hunt verification.

GET/pipeline-handoff.json

Fetch the production pipeline handoff runbook for NTO/NPO and creator workflows.

GET/npo-pipeline-handoff.json

Fetch the focused NPO podcast/audio production handoff for cost, sidecars, media ingress, marker CSV, and repair-loop workflows.

POST/v1/qc/estimate

Preflight cost, margin guardrail behavior, and effective checks before uploading media.

POST/v1/qc/jobs

Create a QC job from a YouTube URL, upload id, or signed asset URL.

POST/v1/qc/jobs/drain

Process queued async QC jobs for Render cron/workflow worker execution.

GET/v1/qc/jobs/{job_id}

Poll job status, verdict, gate progress, and minute usage.

GET/v1/qc/jobs/{job_id}/report

Fetch shareable JSON/PDF-ready defects, timestamps, and evidence.

GET/v1/qc/jobs/{job_id}/events

Inspect lifecycle events for agent explanations and progress audit.

GET/v1/qc/jobs/{job_id}/artifacts

List generated reports, marker exports, clips, and related QC artifacts.

GET/v1/qc/jobs/{job_id}/artifacts/markers

Download editor marker CSV for a completed QC run.

POST/v1/qc/jobs/{job_id}/gate-verdict

Submit an external full-video gate VERDICT.json into UploadCheck.

POST/v1/qc/jobs/{job_id}/cancel

Cancel queued or active work before minutes are finalized.

POST/v1/uploads

Create a signed upload target for agents and self-serve users.

PUT/v1/uploads/{upload_id}/content

Send local media bytes to the signed upload URL before creating an upload_id QC job.

GET/v1/uploads/{upload_id}

Check upload processing and metadata probe status.

GET/v1/qc/jobs?limit=&status=&source_url=

List recent jobs and avoid duplicate agent runs.

GET/v1/usage

List recent usage ledger entries; stored workspace API keys are pinned to their own workspace.

GET/v1/usage/margins?billing_period=&limit=

Summarize usage COGS, allocated revenue, cost per minute, and gross margin.

GET/v1/abuse-events?workspace_id=&limit=

List persisted fail-fast abuse-limit events for operator review.

GET/v1/spend-alerts?workspace_id=&limit=

List persisted Resend overage-spend alert attempts for operator review.

POST/v1/api-keys

Create a hashed workspace API key for paid Claude Code, Codex, Cursor, and MCP clients.

POST/v1/checkout/provision-api-key

Provision a paid checkout customer into an idempotent workspace API key for MCP/API clients.

GET/v1/api-keys?workspace_id=

List redacted API-key records without returning bearer secrets.

UploadCheck is the QC authority. Agents repair only the flagged spans they can reach, then rerun before upload. Checked minutes are deterministic publish-readiness QC minutes.
